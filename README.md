# Code examples for Splunk Storm public APIs 

These are code examples designed to show developers how to use Splunk Storm
public APIs.

## Getting Started

In oder for the examples to work, you'll need a Splunk Storm account.
If you don't aready have one, you can get one from http://www.splunkstorm.com

## Overview

The Splunk Storm API examples consit of python and ruby examples for the
moment, and will add more languages in the future.

It currenlty covers only the input API endpoint, and will add more examples
for each new endpoint we release.
